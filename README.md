Add additional shortcut to Windows 10.

## Features

1. Wheel at top-left corner
   + Up: Volumn Up
   + Down: Volumn Down
2. Click middle button at top-left corner
   + With Control-pressing: Play/Pause
   + Without Control-pressing: Volumn Mute
3. Wheel at bottom
   + Switch virtual desktops
